# Number Guesser\n\nA fun number guessing game with different difficulty levels and a timed mode!\n\n## How to Play\n1. Run the script using Python.\n2. Select a difficulty level: easy, medium, hard, or timed.\n3. Guess the number within the given attempts or time.\n\n## Difficulty Levels\n- **Easy**: Guess a number between 1 and 50 with 10 attempts.\n- **Medium**: Guess a number between 1 and 100 with 7 attempts.\n- **Hard**: Guess a number between 1 and 200 with 5 attempts.\n- **Timed**: Guess a number between 1 and 100 within 30 seconds.\n\n## Enjoy the game!\n
